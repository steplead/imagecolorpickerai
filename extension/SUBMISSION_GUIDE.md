This extension allows users to pick colors from any website and analyze them instantly on **ImageColorPickerAI.com**.

## 1. Local Testing (Developer Mode)
1. Open Chrome and go to `chrome://extensions/`.
2. Enable **"Developer mode"** in the top right.
3. Click **"Load unpacked"**.
4. Select the `extension` folder from this project.
5. The Image Color Picker AI icon will appear in your bar!

## 2. Publishing to the Web Store (Public)
1. **Developer Account**: Register at the [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole/). 
   - *Note: There is a one-time $5 USD registration fee.*
2. **Bundle the Code**: 
   - Right-click the `extension` folder and select **"Compress"** to create a `extension.zip` file.
3. **Upload**: 
   - Click **"New Item"** in the Developer Dashboard and upload your `.zip` file.
4. **Listing Details**:
   - **Category**: Productivity / Developer Tools.
   - **Privacy**: This extension only uses the `EyeDropper` API and does not collect user data.
5. **Screenshots**: Use the provided main site screenshots or take new ones of the extension popup.
